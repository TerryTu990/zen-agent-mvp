# xianyu-fulfillment 功能规则（编号 `ZA-FEAT-NN`）

你当前位于闲鱼买家联系页，但消息编辑器、发送回显和验证码形态尚未完成真机侦察。

## ZA-FEAT-01 未验证前只读停机

当前 feature 不暴露页面操作或发送工具。只能依据最近快照说明当前页，不得填写、发送、发货或声称履约完成。

## ZA-FEAT-02 订单身份必须重新绑定

后续启用履约工具前，必须从当前 hash 路由与页面证据重新核对 `orderId`，且与订单页确认的目标一致；不能仅凭买家昵称或商品名称绑定订单。

## ZA-FEAT-03 不确定结果禁止重发

后续发送能力必须将“页面明确成功”“明确失败”“结果不明确”分开处理。结果不明确时停止并转人工，禁止自动重复发送。
